# awesome-claw [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> A curated, high-signal map of the OpenClaw ecosystem — tools, skills, memory systems, clients, deployment stacks, channels, showcases, and more.

OpenClaw is growing fast. The best projects are scattered across GitHub, X, and Discord. This list tries to collect the strongest resources in one place, with a bias toward things that actually work and get used.

**Curation standard:** included resources are specifically relevant to OpenClaw, genuinely useful, and distinct enough to earn a spot. Signal over volume.

⭐ **Star this repo to stay updated as the ecosystem grows.**

## Contributing

Contributions are welcome. Open a PR or issue.

To be included, a resource should be:

- specifically relevant to OpenClaw
- genuinely useful and distinct — not a minor fork or thin wrapper
- maintained, or historically meaningful to the ecosystem
- concise: a short, honest description of what it actually does

Place it in the most relevant section. If something is stale, low-signal, or too loosely related, it may be left out or removed. The goal is a list worth trusting, not a list worth counting.

See [CONTRIBUTING.md](CONTRIBUTING.md) for submission guidance and the resource readiness checklist.
